# 📚 个人笔记库

欢迎来到我的个人笔记库！这里是我整理知识、记录思考和积累经验的地方。

## 🗂️ 目录结构

```
note/
├── README.md          # 本文件 - 笔记库总览
├── guides/            # 指南和教程
├── projects/          # 项目笔记
├── learning/          # 学习笔记
├── references/        # 参考资料
├── daily/             # 日常记录
├── templates/         # 笔记模板
├── scripts/           # 整理脚本
├── system/            # 系统配置文档
└── archive/           # 归档文件
```

## 📖 使用指南

### 1. 快速开始
- 查看 `templates/` 目录获取笔记模板
- 按照主题将笔记放入对应目录
- 定期使用 `scripts/` 中的工具整理笔记

### 2. 命名规范
- 文件使用英文小写，单词用连字符分隔
- 日期格式：YYYY-MM-DD
- 示例：`2024-03-08-git-commands.md`

### 3. 标签系统
在笔记顶部使用 YAML front matter 添加标签：
```yaml
---
title: "笔记标题"
tags: [tag1, tag2]
date: YYYY-MM-DD
---
```

## 🔄 维护流程

1. **每日**：在 `daily/` 中添加当日记录
2. **每周**：整理本周笔记，归档到对应目录
3. **每月**：回顾并清理旧笔记
